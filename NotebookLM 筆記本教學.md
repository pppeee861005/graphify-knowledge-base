這是一份根據影片內容整理的結構化筆記，介紹如何透過 **Hermes Agent 集成 NotebookLM** 構建超級知識庫：

### 1) 影片主題概述
本影片介紹如何將 Google 的 **NotebookLM 無縫集成至 Hermes Agent**，打造一個具備強大記憶與多媒體生成能力的個人 AI 超級知識庫。這種整合方式能解決傳統 RMWK 方案 Token 消耗大、多設備同步不便的問題，並能自動生成投影片、視頻與播客，顯著提升學習與研究效率。

### 2) 主要功能介紹
*   **自然語言操控**：直接用自然語言控制 NotebookLM 進行筆記列出、切換、內容查詢等操作。
*   **多媒體內容生成**：可自動生成 **幻燈片 (PPT)**、**講解視頻**、**心智圖 (MindMap)** 及 **測驗卷 (Quiz)**，並直接保存至本地桌面。
*   **自動化論文處理**：支援自動搜素論文、創建新筆記本、上傳 PDF 並進行深度研究講解。
*   **多設備同步與分享**：所有操作與網頁版同步，生成的筆記本可直接分享給其他用戶。

### 3) 操作步驟
1.  **安裝組件**：在 Hermes Agent 中輸入優化後的開源項目 `notebook-rm-py` 倉庫鏈接，讓 Agent 自動完成安裝。
2.  **帳號登錄**：透過終端機執行命令進行帳號驗證，或使用 Cookie 導出方式實現登錄（作者優化版已修復 Cookie 易過期的問題）。
3.  **發送指令**：直接向 Agent 下達指令，例如「將這篇論文添加到筆記」或「根據這三篇論文生成心智圖」。
4.  **結果獲取**：Agent 會調用對應 Skill 執行任務，並將生成的結果（文本或文件）返回給用戶。

### 4) 關鍵技巧
*   **跨語言查詢優化**：Agent 會自動將用戶輸入的中文提示詞翻譯並擴充為更精準的**英文查詢指令**，以獲得更詳盡的檢索結果。
*   **本地自動保存**：生成的 PPT、視頻或心智圖文件可設定直接存放在本地桌面，方便即時調用。
*   **技能擴展**：透過集成的 Skill，Agent 能自動處理複雜的 Python 包調用與 Cookie 刷新，無需用戶手動干預。

### 5) 總結
Hermes Agent 與 NotebookLM 的結合，為用戶提供了強大的「**知識庫外掛**」與「**知識飛輪**」。這套系統不僅能高效管理海量資訊，還能將碎片化的知識轉化為結構化的多媒體產出，是極致的個人 AI 知識庫解決方案。
