## 使用方式

### 純文字搜尋（search_files）

```bash
search_files --pattern "關鍵字" --target content --file_glob "*.md"
```

### 知識圖查詢（graphify query）

```bash
graphify query "查詢內容" --graph /path/to/graph.json
```
