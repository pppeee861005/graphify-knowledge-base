## 第三部分：變現系統設置

### 1. 啟用 Substack Pro（如果還沒有）

**位置：** Settings → Paid Subscriptions

- 啟用付費訂閱
- 設置月費：$10 USD（或 ¥80 CNY）
- 設置年費選項：$100 USD（或 ¥800 CNY，鼓勵年訂）

### 2. 設置付費專欄內容

**首先，決定內容策略：**

**選項 A：混合模式（推薦）**
```
Free 訂戶看到：
└─ 每周四：核心文章（完整版，無縮短）

Pro 訂戶額外看到：
├─ 每周一：「內部信」- 市場觀察 + 個人洞察
├─ 每月：「Skill 技術棧」- 新工具教學
└─ 每月：「社群報告」- 新人類聯盟數據洞察
```

**選項 B：分層內容**
```
Free：摘要版（1500 字）
Pro：完整版（3000 字）+ 補充資源
```

**我的建議：用選項 A**
- 不要縮短 Free 版本（傷害信任）
- 用新的專欄內容吸引升級
- 讓 Free 和 Pro 都感到有價值

### 3. 發布第一篇 Pro-only 文章

**時機：** 獲得 50+ 訂閱者後

**範例標題：**

```
內容大綱：
1. 背景：每周浪費 20 小時在重複任務
2. 解決方案：構建的 3 個 Agent
3. 成果：時間節省了 50%，質量提升了 30%
4. 技術細節：Hermes + Skill + CubeSandbox
5. 你可以複製的模板：[下載鏈接]

預期：
- 展示你真正有價值的東西
- 讓 Pro 訂戶感到「值回票價」
```

**位置：** Settings → Sponsorships

**計畫贊助對象：**
1. **Cursor** - 如果你經常推薦它

## 常見陷阱 ⚠️

❌ **不要做的事：**

1. **隱藏 Free 版本內容**
   - 只會激怒讀者，降低信任度
   - 合適做法：發布完整 Free 版本 + Pro 延伸

2. **過度推銷**
   - 每篇文章都推 Pro 會讓人厭煩
   - 合適做法：1/3 文章有明確 CTA，其他隱含

3. **忽視郵件打開率**
   - 主旨很重要！浪費了
   - 測試不同主旨：
     - 「我做了什麼就增長了 10 倍」(高點擊)
     - 「這個 Skill 模板價值 $500」(高轉化)
     - 「周報：5 月市場觀察」(信息類)

4. **發帖不一致**
   - 訂閱者期待穩定的節奏
   - 承諾周四發，就周四發

5. **忽略讀者反饋**
   - 回覆郵件是建立關係的黃金機會
   - 每週花 1 小時回覆讀者
