
---

## 🏗️ 四層 Agent 架構

| 層級 | 功能 | 技術 |
|------|------|------|
| **感知** | 天氣、日程、衣柜 | API集成 |
| **決策** | 推薦搭配方案 | Claude LLM |
| **執行** | 虛擬穿衣圖 + 購物 | Kling API |
| **反饋** | 用戶評分 → AI學習 | 數據反饋循環 |

---

## 💰 商業模式

```
免費版    → 1套推薦/天（廣告變現）
Pro版     → $9.99/月（5套推薦+虛擬穿衣）
企業版    → $499–2,999/月（白牌化）
```
