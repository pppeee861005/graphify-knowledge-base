## 🔧 遇到的問題與解決

### 問題 1：Perplexity API 認證失敗
**錯誤訊息：** `PERPLEXITY_API_KEY environment variable not set`

**原因：** .env 文件使用 `perplexity=` 而非標準 `PERPLEXITY_API_KEY=`

**解決方案：**
```python
with open(env_path, 'r', encoding='utf-8') as f:
    for line in f:
        if 'perplexity=' in line:
            api_key = line.split('=', 1)[1].strip()
client = Perplexity(api_key=api_key)
```
