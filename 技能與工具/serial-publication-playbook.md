# Serial Publication Playbook
## 連載發布計劃編排技巧守則（供 AI 執行參考）

**適用場景**：Substack / 部落格 / LinkedIn 長篇系列文章規劃
**設計目標**：讓讀者像對待連續劇一樣追更，無法中途放棄
**版本**：1.0（從《逆向視覺解構》系列逆向萃取）

---

## PART 1：系列架構設計守則

### 守則 01｜先定情緒曲線，再定篇數

不要先想「我有幾個觀點」，要先問「讀者的情緒應該經歷哪些階段」。

標準情緒波形如下，依序排列：

```
震撼切入 → 理論鉤子 → 恐懼引爆 → 多巴胺高峰 → 哲學升維 → 行動收官
```

每個階段至少一篇。高峰期（多巴胺）可安排 1–2 篇，是瘋傳引擎。
篇數由情緒波形決定，不由觀點數量決定。

---

### 守則 02｜核心命題必須是「典範轉移」級別

系列的核心命題不能是觀點，必須是衝擊世界觀的宣告。

判斷標準：
- ✅ 讀者看完後，對某件事的理解永久改變
- ✅ 命題可以用一句話說清楚，但需要八篇才能說服
- ❌ 讀者看完後只是「知道了一件事」

範例：
> ❌ 弱命題：「AI 可以幫助建築師畫圖」
> ✅ 強命題：「當 AI 學會從成品反推設計，實體世界的所有 Know-How 都將被重新分配」

---

### 守則 03｜第一篇必須有「驚嚇瞬間」場景

開篇不能從理論開始。必須從一個具體的、讓人產生「等等，這是真的嗎」的場景開始。

格式：
```
[我做了一個動作] → [AI 還我一個讓人震驚的結果]
```

場景選擇標準：
- 動作要極度日常（丟一張照片、輸入一個名字）
- 結果要極度不對稱（出來施工大樣圖、出來完整購買清單）
- 這個不對稱感必須讓任何行業的人都能感受到威脅

---

### 守則 04｜戰略定位篇不放中間

如果系列中有一篇是「我的產品/定位/商業模式」，不能放在中段。

原因：讀者正在享受思想之旅，中段切換成自我介紹會讓情緒溫度驟降。

正確做法：
- 將戰略意圖折疊進收官篇（最後一篇）
- 在收官篇用「武器庫公開」的方式自然帶出產品定位
- 順序：先完成讀者的思想旅程，再說「所以我的位置是這裡」

---

### 守則 05｜瘋傳篇放在系列中段（W4–W5 位置）

瘋傳篇的特徵：
- 打向最廣大的普通讀者（不需要專業背景也能震撼）
- 有情緒張力（慾望、恐懼、感動三選一）
- 有即時的現實連結（看得到、摸得到、買得到）

瘋傳篇放中段的理由：
- 前段建立信任，讓核心讀者願意分享
- 中段瘋傳引入新讀者，新讀者從頭追起
- 後段新老讀者一起走向收官，轉化率最高

## PART 2：篇章設計守則

### 守則 06｜每篇必須有獨立的「情緒任務」

每篇文章不只有資訊任務，還有情緒任務。寫作前先定義兩者。

| 欄位 | 定義 |
|---|---|
| 資訊任務 | 讀者讀完後知道了什麼 |
| 情緒任務 | 讀者讀完後感受到什麼 |

情緒任務選項：
- 震撼（「這是真的？」）
- 恐懼（「我的行業會是下一個嗎？」）
- 慾望（「我也想要這個能力」）
- 感動/不安（「連這個都可以被解構？」）
- 憤怒（「為什麼沒人告訴我」）
- 完成感（「我走完了一段旅程」）

---

### 守則 07｜案例選擇遵循「理性 → 感官 → 美學 → 心理」遞進

當一篇文章需要多個案例，按以下順序排列，最後一個案例打擊力最強：

```
理性案例（物質層）→ 感官案例（身體層）→ 美學案例（品味層）→ 心理案例（情感層）
```

範例：
```
建築照片 → 施工圖（理性）
料理成品 → 食譜（感官）
服裝單品 → 版型（美學）
電影大片 → 情感觸發機制（心理）← 壓軸，最重
```

最後一個案例必須讓讀者意識到：「連這個都可以被解構」，這才是真正的典範轉移。

---

### 守則 08｜新增章節的引入條件

在原始規劃外新增章節，必須同時滿足以下三個條件：

1. **覆蓋新受眾**：原有章節未能觸及的讀者群
2. **提供新商業閉環**：有清晰的變現路徑（Affiliate / 訂閱 / Skill 銷售）
3. **強化核心命題**：不是補充，是讓命題更有爆炸性

不滿足以上三點的內容，折疊進現有章節，不單獨成篇。

## PART 3：結尾與預告詞設計守則

### 守則 09｜結尾收官語三個禁忌

結尾不是總結。以下三種寫法全部錯誤：

```
❌ 總結型：「本文介紹了 X、Y、Z 三個概念」
❌ 呼籲型：「希望大家能夠重視這個趨勢」
❌ 預告型：「下週我們將介紹...」（預告詞是獨立區塊，不混入收官語）
```

正確的收官語格式：**炸彈引爆後的煙霧**

特徵：
- 一句話，不超過兩句
- 讓讀者帶著一個「未完全消化的震撼」離開
- 這個震撼在腦中繼續發酵，到下週四才被下一篇解除

範例模板：
```
[本篇核心衝擊的高度壓縮] + [開放性的懸念或哲學質問]
```

---

### 守則 10｜預告詞的雙句結構

每篇結尾的預告詞由兩句構成，缺一不可：

**第一句：懸念攻擊**
- 格式：一個讓人產生「等等，什麼？」的陳述或問句
- 不解釋，只挑釁
- 攻擊的對象是讀者的信仰或預設（不只是好奇心）

**第二句：輪廓給予**
- 格式：「下週四：〈篇名〉——[一句話說清楚下篇打什麼]」
- 給出足夠的輪廓，讓讀者能預想「下週四我會學到什麼」
- 絕對不洩底，不給結論，只給框架

範例：
```
第一句（懸念攻擊）：
我們以為情感是最後的聖域。那是我們最大的誤解。

第二句（輪廓給予）：
下週四：〈連「感動」都可以被反編譯〉——AI 解構一部讓你哭的電影，找出你哭的精確原因。
```

---

### 守則 11｜最強預告詞攻擊「信仰」而非「好奇心」

好奇心預告：「下週我們會介紹一個你可能不知道的技術」→ 弱
信仰攻擊預告：「我們以為情感是最後的聖域。那是我們最大的誤解」→ 強

判斷方式：讀者看完第一句，是「哦，有趣」還是「等等，這不對吧」？
後者才是有效預告。

信仰攻擊的常見切入點：
- 「我們一直以為 X 是安全的。它不是。」
- 「你以為只有 X 會被影響。你低估了它。」
- 「我們以為 X 是人類獨有的。那是最後的誤解。」

---

### 守則 12｜收官篇預告詞使用「稀缺感」而非「懸念」

最後一篇的預告詞邏輯不同。讀者已不需要懸念，需要的是「這是最後機會」的稀缺感。

格式：
```
[系列終章宣告] + [武器庫/工具/資源的具體承諾] + [訂閱行動引導]
```

範例：
```
系列終章。武器庫全部公開。Super Individual 的核武器，這週交到你手裡。

下週四收官：〈反編譯實體世界〉——看建築出施工圖、看電影出情感公式。工具、Prompt、Skill 全部釋出。
```

## PART 4：時間折疊守則（收官設計）

### 守則 13｜收官篇收官語必須折回第一篇

收官語的最高技法：時間折疊。

把讀者的記憶拉回系列第一篇的具體場景，讓整個系列形成閉環。

效果：讀者感受到「我走完了一段旅程」，這種完成感會驅動他們：
1. 主動分享這個系列
2. 向新朋友推薦「從第一篇開始看」
3. 期待下一個系列

格式：
```
[時間錨點：X 週前你看到/讀到/做了什麼] + [現在你知道那意味著什麼] + [這不是終點，是起點]
```

範例：
```
八週前，你看到一張陽台欄杆的照片。今天，你知道那張照片意味著什麼。
這不是終點——這是你開始反編譯世界的第一天。
```

## PART 5：系列副產品設計守則

### 守則 14｜每個系列必須有「武器庫」作為留存鉤子

武器庫是讀者訂閱後能獲得的具體資源，必須在收官篇公開。

武器庫構成標準（選 2–3 項）：

| 類型 | 說明 | 轉化效果 |
|---|---|---|
| Prompt 模板庫 | 對應系列主題的可直接使用 Prompt | 高（立即實用） |
| Skill 模組 | 可部署的 AI Skill 檔案 | 極高（技術門檻創造稀缺） |
| GitHub Repo | 開源版本，建立社群 | 中（長尾流量） |
| Podcast/音頻 | NotebookLM 解說版 | 中（不同媒介觸達） |
| PDF 工具書 | 系列精華壓縮版 | 高（可分享的實體感） |

武器庫在系列進行中可以漸進式預告（「W8 武器庫全公開」），製造稀缺期待。

---

### 守則 15｜副產品命名遵循動詞原則

副產品的名稱必須是動詞或動詞短語，不能是名詞描述。

```
❌ 名詞描述：「逆向解構工具集」
✅ 動詞短語：`reverse-blueprint`（照片轉施工大樣圖）
✅ 動詞短語：`film-deconstructor`（電影情感機制解析）
```

動詞命名的原因：讓讀者一眼看懂「這個工具幫我做什麼」，降低認知摩擦，提高點擊率。

---

## PART 6：AI 執行時的優先級排序

當接到「規劃一個 Substack 系列」的任務，按以下順序執行：

```
Step 1：確認核心命題（必須是典範轉移級別）
Step 2：設計情緒曲線（六段式波形）
Step 3：定義瘋傳篇（中段，打向最廣大受眾）
Step 4：確認戰略定位篇位置（折疊進收官篇）
Step 5：逐篇定義情緒任務（不只是資訊任務）
Step 6：撰寫每篇收官語（炸彈煙霧格式）
Step 7：撰寫每篇預告詞（懸念攻擊 + 輪廓給予）
Step 8：設計收官篇時間折疊句
Step 9：規劃武器庫副產品清單
```

---

## 快速檢查清單（發布前自查）

```
□ 核心命題是典範轉移級別，不是觀點級別
□ 第一篇有具體的「驚嚇瞬間」場景
□ 情緒曲線完整（六段波形全部覆蓋）
□ 瘋傳篇在 W4–W5 位置
□ 戰略定位內容在收官篇，不在中段
□ 每篇都有獨立的情緒任務（非只有資訊任務）
□ 每篇收官語是煙霧格式（不是總結）
□ 每篇預告詞有雙句結構（懸念攻擊 + 輪廓給予）
□ 最強預告詞攻擊信仰，不只是好奇心
□ 收官篇預告詞使用稀缺感邏輯
□ 收官語折回第一篇場景（時間折疊）
□ 武器庫副產品清單完整，命名使用動詞原則
```
