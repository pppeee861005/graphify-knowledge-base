如何使用這份範本

1. 複製貼上至你偏好的編輯器（例如 ~/ai-commander-hq/🧠學習備忘錄/每日日誌_YYYYMMDD.md），日期與時間自行填寫。
2. 依實際情況填寫：若當天沒有某一區塊（例如「關鍵數據」），可直接刪除或保留空白。
3. 完成後：
  • 可用 read_file 檢視，或直接 git add/commit（若你有版控）。
  • 若想納入知識圖，執行 graphify add /path/to/每日日誌_YYYYMMDD.md --workspace ~/graphify_learning（之後即可免費用 graphify query 檢索關鍵字）。
4. 長期累積：所有每日日誌匯入同一個 Graphify 工作區後，你就能用 graphify query "token 使用"、graphify explain "錯誤處理機制" 等快速追蹤歷史趨勢。

│ 小提醒：若你更喜歡分開寫「工作日誌」「學習紀要」「今日成就」，也可以保留你原來的三個檔案，只是在每日結束時用一個簡單的腳本（或 daily-learning-update Skill）把它們彙總成上述單一檔案，以後查詢會更方便。
