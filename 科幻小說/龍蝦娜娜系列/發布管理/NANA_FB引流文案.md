## 💡 進階技巧

### 1. 貼圖搭配
```
使用電影預告海報 Prompt 生成的圖片
+ 高對比版本（紅金碰撞）
+ 清晰顯示兩個角色
= 視覺點擊率 +300%
```

### 2. 表情符號策略
```
✅ 使用 = 品牌識別（每篇都用龍蝦🦞）
✅ 使用顏色符號 = 視覺層次（🔴🟡 代表角色）
✅ 使用方向符號 = 行動號召（👉 指向連結）
❌ 避免過度 = 最多8-10個符號/篇
```

### 3. 文案長度黃金比例
```
短文案：20-30字（吸睛）
+ 中文案：40-60字（說故事）
+ 長文案：60-100字（深度）
= FB 最佳混合長度
```

### 4. CTA（行動號召）強度
```
弱 CTA：「看看吧」、「有興趣的點進來」
中 CTA：「故事在這裡 👉」、「連結在下面」
強 CTA：「5月18日限定發布，錯過就沒有」、「免費看」
最強 CTA：「在留言區告訴我你會選A、B還是C」
```
