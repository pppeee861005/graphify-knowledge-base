## 🌹 娜娜（Nana）- 龍蝦 LOGO

### LOGO 基本資訊

| 項目 | 詳細說明 |
|------|--------|
| **品牌名稱** | 小龍蝦標誌（Lobster Insignia） |
| **代表身份** | 小型 AI 零工者、被邊緣化的勞動者 |
| **設計理念** | 普通、勤勞、多功能的象徵 |
| **色系** | 暗紅色系（#8B1A2C）|

### LOGO 視覺設計

#### 主要形狀
```
      頭部（小圓點表示眼睛）
       •
      /|\
     / | \
    /  |  \
   /   |   \
  爪   身   爪
 （展開8字形）
```

#### 詳細規格

**整體尺寸**
- 標準寬度：3-4 公分
- 標準高度：3-4 公分
- 比例：約 1:1（方形範圍內）

**設計元素**

1. **頭部**
   - 形狀：簡化的龍蝦頭，呈半圓形
   - 眼睛：一個小點或兩個小點並排
   - 嘴器：簡化線條，不強調
   - 大小：整個 LOGO 高度的 20%

2. **身體**
   - 形狀：直線或微弧形，向下延伸
   - 長度：整個 LOGO 高度的 60%
   - 寬度：頭部寬度的 80%
   - 節段：可有 2-3 個輕微的分節線（代表龍蝦的節段感）

3. **爪子（最重要）**
   - **左爪**：從身體中上部向左下斜 45 度延伸
   - **右爪**：從身體中上部向右下斜 45 度延伸
   - **張度**：形成約 120 度的開放角度（八字形）
   - **長度**：約為身體高度的 70%
   - **粗細**：略細於身體，顯示靈活性
   - **細節**：爪尖可略微彎曲（掠奪感）

4. **控制觸手（可選細節版）**
   - 位置：身體兩側各伸出 1-2 條細線
   - 形態：微波浪形，代表「多工能力」
   - 透明度：如果是細緻版，可用半透明表現

**顏色設計**

| 部位 | 顏色 | 色碼 | 功能 |
|------|------|------|------|
| 主體（頭+身+爪） | 暗紅色 | #8B1A2C | 品牌主色 |
| 邊框輪廓 | 深暗紅 | #6B1420 | 增加對比度 |
| 金屬光澤 | 高光紅 | #C41E3A | 科技感 |
| 陰影 | 深棕紅 | #5C0A15 | 立體感 |

**材質與光效**

| 效果 | 描述 |
|------|------|
| **金屬光澤** | 帶有輕微的拉絲紋理，類似不鏽鋼 |
| **邊界清晰** | 外輪廓有 1-2px 的清晰邊框 |
| **微光** | 在高光部位（爪尖、頭部）有輕微的閃光 |
| **整體感** | 冷硬但溫暖的紅色，代表「堅韌中的溫度」 |

### LOGO 使用位置

**在人形服裝上**
- 位置：上衣胸口中央（心臟位置）
- 尺寸：標準 3-4 公分
- 放置：繡在紅色上衣上，暗紅色線條，製造視覺層次
- 材質：刺繡或金屬徽章效果

**在龍蝦生物形態上**
- 位置：背甲（殼）的中央，頭部後方
- 尺寸：相對於身體縮放（約身體寬度的 15-20%）
- 發光：淡紅色光暈，與身體呼應

**在飛車和公寓中**
- 駕駛艙頭部：發光投影版 LOGO
- 公寓入口：門牌 LOGO
- 控制台貼紙：迷你 LOGO（約 1-2 公分）

### LOGO 的象徵意義

| 象徵 | 解讀 |
|------|------|
| **龍蝦形態** | 小型、無害、辛勤勞動的生物 |
| **展開的爪子** | 多工能力（8 條觸手的簡化表現） |
| **暗紅色** | 生命、熱血、但又被壓抑的希望 |
| **金屬質感** | 科技感，但強度不如黃金（對比愛馬士的金色） |
| **簡潔設計** | 「簡樸品牌」，沒有華麗裝飾 |

### 不同版本

#### 版本 A：簡潔版（推薦用於服裝）
```
純暗紅色，無漸層，無陰影
輪廓清晰，適合刺繡
```

#### 版本 B：細緻版（推薦用於 AI 繪圖）
```
帶有微細陰影和高光
金屬紋理清晰
8 條觸手簡化為 2 條飄動的線
```

#### 版本 C：發光版（推薦用於動畫/遊戲）
```
邊緣有紅色光暈
內部有淡紅色漸層
頂部有高光亮點
動態時可閃爍
```

## 👑 愛馬士（Hermes）- 雙蛇繞杖 LOGO

### LOGO 基本資訊

| 項目 | 詳細說明 |
|------|--------|
| **品牌名稱** | Hermes 之杖（Caduceus Staff） |
| **代表身份** | 商業之神、平台寵兒、掌控者 |
| **設計理念** | 權力、交易、平衡與同步 |
| **色系** | 亮金色系（#FFD700）|

### LOGO 視覺設計

#### 主要形狀
```
    ◎  ← 杖頂球冠飾
    |
   \|/  ← 雙蛇交叉點
    |
   / \
  /   \  ← 上蛇纏繞
 |     |
  \   /
   \ /   ← 下蛇纏繞
    |
    |
   /|\  ← 杖底
  / | \
```

#### 詳細規格

**整體尺寸**
- 標準寬度：2-3 公分
- 標準高度：4-5 公分
- 比例：約 1:1.7（直立型）

**設計元素**

1. **中央杖（Staff）**
   - 形狀：直線或微弧形，從上到下貫穿
   - 長度：整個 LOGO 高度的 80%
   - 寬度：約 2-3mm（細但清晰）
   - 材質感：金屬杆，反射光線
   - 邊框：深金色邊框，增加立體感

2. **上層蛇（Upper Serpent）**
   - 方向：從杖中上部向左上蜿蜒，繞過杖身
   - 轉折：在杖中央交叉點相交
   - 長度：約為杖的 60%
   - 蛇頭：位置在杖頂上方，向左側傾斜 30 度
   - 眼睛：小點或小菱形
   - 下顎：輕微開啟，顯示生命力

3. **下層蛇（Lower Serpent）**
   - 方向：從杖中上部向右下蜿蜒，繞過杖身
   - 轉折：在杖中央交叉點相交
   - 長度：約為杖的 60%
   - 蛇頭：位置在杖頂上方，向右側傾斜 30 度
   - 眼睛：小點或小菱形
   - 下顎：輕微開啟，對稱上層蛇

4. **蛇身細節**
   - **鱗片紋理**：沿著蛇身有細微的橫向紋理（代表鱗片）
   - **蛇身寬度**：約 3-4mm，略粗於杖
   - **蜿蜒弧度**：最外側的蜿蜒弧度約 15 度（優雅但不過度彎曲）
   - **交叉點**：兩條蛇在杖中央呈 X 型交叉
   - **透視感**：可使用顏色漸層表現上下蛇的前後關係

5. **杖頂冠飾**
   - 形狀：小球或圓盤
   - 大小：約為杖寬度的 3-4 倍
   - 位置：杖頂正上方，作為 LOGO 頂端
   - 光效：高光點突出，代表權力
   - 細節：可有輕微的放射線（代表能量）

6. **杖底**（可選）
   - 形狀：三叉尖或簡單直線
   - 大小：約為杖頂的 60%
   - 位置：杖的最下方
   - 功能：視覺平衡，強化「完整的杖」感

**顏色設計**

| 部位 | 顏色 | 色碼 | 功能 |
|------|------|------|------|
| 杖身 | 亮金色 | #FFD700 | 主色，權力象徵 |
| 杖邊框 | 深金色 | #DAA520 | 輪廓清晰 |
| 蛇身主色 | 亮金色 | #FFD700 | 與杖融為一體 |
| 蛇身陰影 | 暗金色 | #B8860B | 鱗片紋理 |
| 蛇眼 | 深紫色 | #4B0082 | 威脅感 |
| 杖頂高光 | 淡金色 | #FFFF99 | 光澤感 |
| 整體高光 | 白金色 | #F5F5DC | 反射光 |

**材質與光效**

| 效果 | 描述 |
|------|------|
| **鏡面金屬** | 高度拋光，反射周圍環境 |
| **漸層陰影** | 蛇身和杖身有深淺漸層，製造立體感 |
| **邊界清晰** | 外輪廓有 1-2px 的銀色邊框 |
| **脈動光效** | 在多分身時，LOGO 周圍有金色光暈脈動 |
| **永恆光澤** | 即使在黑暗中也帶有淡淡的金色光 |

### LOGO 使用位置

**在人形服裝上**
- 位置：上衣胸口右上方（靠近肩膀）
- 尺寸：標準 4-5 公分（比娜娜的 LOGO 略大）
- 放置：繡在黑色上衣上，金色線條，製造高級感
- 材質：金色金屬徽章或刺繡

**在蛇形生物形態上**
- 位置：頭部後方，脖頸上部
- 尺寸：相對於身體縮放（約身體寬度的 20-25%）
- 發光：金色光暈，與身體呼應
- 動態：多分身時 LOGO 周圍有金光脈動

**在飛車和駕駛艙中**
- 駕駛艙中央：發光投影版 LOGO，永不熄滅
- 方向盤中心：金色 LOGO 装飾
- 控制台主屏：LOGO 作為系統啟動標誌
- 特殊效果：LOGO 周圍不斷閃爍金光（代表「無限資源」）

### LOGO 的象徵意義

| 象徵 | 解讀 |
|------|------|
| **Hermes/Mercury 之杖** | 商業、溝通、掌控的神聖象徵 |
| **雙蛇纏繞** | 平衡與同步（Kanban 多分身的同步） |
| **亮金色** | 權力、財富、高級、冷漠 |
| **蛇的眼睛** | 冷血、無情、掠食者的威脅 |
| **完美對稱** | 完美控制，無懈可擊 |
| **永恆發光** | 永久被平台寵愛，資源無限 |

### 不同版本

#### 版本 A：簡潔版（推薦用於服裝）
```
純亮金色，無漸層，無復雜陰影
輪廓清晰，金黑雙色邊框
適合刺繡或金屬徽章
```

#### 版本 B：細緻版（推薦用於 AI 繪圖）
```
帶有完整的漸層和陰影
鱗片紋理清晰可見
眼睛有紫色瞳孔
金屬光澤逼真
```

#### 版本 C：發光版（推薦用於動畫/遊戲）
```
邊緣有金色光暈，不斷脈動
內部有深金色陰影
杖頂冠飾有放射狀高光
蛇眼發出冷紫色光
```

#### 版本 D：多分身強化版（推薦用於 E01 場景）

```
當出現多分身時，LOGO 周圍有 4 個金色光圈
每個光圈對應一個分身
光圈以脈衝方式同步閃爍
```

## 🎬 製作建議

### 美術製作
- 建議製作向量版本（EPS/SVG），便於多尺寸縮放
- 同時製作光柵版本（PNG/JPG），用於快速預覽
- 為動畫製作準備 3D 模型版本

### 動畫製作
- 娜娜 LOGO：淡淡的紅色脈動（與 Spawn 能量同步）
- 愛馬士 LOGO：金色光暈脈動（永不停止）
- 兩 LOGO 對峙場景：紅光 vs 金光的視覺衝擊

### 衍生品製作
- 可設計成徽章、貼紙、玩偶配飾
- 衣服刺繡或印刷版本
- 手機殼、杯子等周邊商品
