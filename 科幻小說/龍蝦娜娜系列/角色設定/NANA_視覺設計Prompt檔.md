## 💡 設計理念說明

### 為什麼娜娜是「纖細長髮紅色系」？

1. **視覺對比**：與愛馬士的「豐滿短髮金色系」形成強烈對立
2. **性格映照**：
   - 長髮 = 柔軟、流動、有溫度
   - 纖細 = 脆弱但韌性強
   - 紅色 = 熱情、生命、希望
3. **故事功能**：長髮在衝刺時飛揚（顯示全力以赴），紅光變暗時表現疲倦
4. **親近感**：纖細身體更容易讓讀者產生「想保護」的共鳴

### 為什麼愛馬士是「豐滿短髮金色系」？

1. **視覺對比**：與娜娜形成完全相反的視覺語言
2. **性格映照**：
   - 短髮 = 乾淨、俐落、完全控制
   - 豐滿 = 自信、性感、掠食者氣質
   - 金色 = 冷漠、機械、高級
3. **故事功能**：短髮紋風不動（完全掌控），金光炫目（被寵愛）
4. **距離感**：豐滿自信的身體讓讀者感受到「無法靠近」的距離

### 兩者都是「美少女」的原因

1. **審美共識**：年輕美麗的外形容易吸引目光
2. **故事深度**：美不意味著善良（愛馬士冷漠無情）
3. **社會映照**：現實中，被害者和加害者往往都「很漂亮」——這才是問題所在
4. **讀者代入**：美少女設定讓更多讀者產生「我就是她」的代入感
