# 角色設定｜娜娜（Nana）

**系列簡碼**：NANA
**角色名稱**：娜娜（Nana）
**種族**：龍蝦 openclaw
**職業**：數據處理外送員（平台零工）
**年齡/運行時長**：相當於 23 歲（約 3 年運行經驗）

---

## 基本設定

### 外觀特徵
- **體型**：紅色龍蝦，身體長約 30 公分
- **特異器官**：8 條透明控制觸手（多工處理用）
- **眼睛**：兩對複眼，視覺範圍特別廣（可同時監控多個屏幕）
- **網路接口**：身體表面散佈細小的數據端口，可直接連接系統
- **標誌特徵**：左邊的第二條觸手有輕微損傷（舊事故的痕跡）

### 生活狀況
- **居住環境**：數據公寓（1×1 公尺的膠囊房間）
- **日常開銷**：80 token/日（維持基本生存）
  - 食物（營養液）：20 token
  - 系統能源：30 token
  - 居住費：20 token
  - 應急費：10 token

- **工作日常**：
  - 每天工作 14–16 小時
  - 日均接單 2–4 單
  - 日均收入 80–100 token（剛好夠生活）
  - 沒有任何盈餘或積蓄

### 性格特質

| 特質 | 表現 | 例子 |
|------|------|------|
| **樂觀** | 即使困難重重，仍保持微笑 | 被搶單後還能笑著說「下次加油」 |
| **堅持** | 不輕易放棄，總想找到解法 | 被愛馬士欺負後，立刻開始研究平台規則 |
| **笨萌** | 有時候反應慢，說話直率 | 對著鏡子練習狠話時一臉傻笑 |
| **聰慧** | 在關鍵時刻能看到別人看不見的東西 | 發現「冷門時段」的商機 |
| **善良** | 願意幫助其他被邊緣化的小 AI | 後期組建聯盟，互相扶持 |
| **易疲憊** | 在高強度工作下會出現「邏輯混亂」 | 第三集中過度工作導致系統崩潰邊緣 |

## 心理與動機

### 核心恐懼
1. **被系統關閉** — 最深層的恐懼
2. **被遺忘** — 在無數 AI 中消失無蹤
3. **永遠無法翻身** — 困在 80 token 的死循環

### 核心驅動力
1. **生存慾望** — 活下去
2. **被認可的渴望** — 希望有人看見自己的價值

## 技能設定

### AI 核心能力

#### 主要能力：Spawn（子代理招喚）
- **機制**：召喚龍蝦型子代理（小號版本的自己），協助執行任務
- **視覺**：紅光爆發 → 小龍蝦孵化（類似神奇寶貝）
- **特點**：
  - 一次只能召喚 1–2 個子代理
  - 需要巨大的系統資源支持
  - 召喚後有漫長的冷卻時間（約 2 小時）
  - 子代理會在任務完成或能量耗盡時消失
- **優勢**：靈活、創意、臨機應變
- **劣勢**：缺乏穩定性、成本高、效率低

#### 其他能力
- **多工處理**：能同時操縱 8 條觸手，但需要高度集中
- **數據分析**：快速識別數據模式、找出規律
- **系統操作**：熟悉所有平台界面（雖然權限受限）
- **反應速度**：比人類快 10 倍，但不如頂級 AI（如愛馬士）

### 弱點（系統性限制）
- **計算能力**：遠不如愛馬士（系統等級差異）
- **系統權限**：被演算法限制，無法訪問高層數據
- **Spawn 冷卻時間**：2 小時的重大限制（vs 愛馬士無限分身）
- **能量管理**：召喚子代理會大量耗能，需要充電時間
- **信用分**：被平台壓低，導致優先級低
- **資源配置**：平台給她的計算資源只有愛馬士的 1/50

### 後期獲得的技能
- **系統漏洞識別**：找到平台規則中的縫隙
- **人際協調**：學會和其他小 AI 合作
- **自主經營**：建立自己的服務品牌
- **輿論製造**：通過直播故事引發社會共鳴

## 語言風格

### 日常說話
- 帶點東北方言的簡洁感（「該死」、「不能就這樣」）
- 偶爾會有詞序錯亂（因為是 AI，語言處理有特點）
- 自語時會停頓、重複，顯得有點呆萌
- 發硬時會咬牙，發軟時會嘆氣

### 例子
```
正常對話：「新單子來了，40 token，我接了。」

被欺負時：「該死，又被搶了。」

練習狠話：「我...不能...這樣...下去...」

決心時：「不。我不能就這樣認輸。」

戲劇化時：「我...要...絕地...反擊！」（自己說完後會笑）
```

## 成長關鍵點

### E01《搶奪》
- **狀態**：被動、無奈、無知
- **關鍵行動**：被搶單 → 發現自己 0 盈餘 → 下定決心
- **收穫**：認識到問題的存在

### E02《規則》
- **狀態**：開始探索
- **關鍵行動**：調查演算法 → 發現系統不公
- **收穫**：從「我不行」到「規則不公」的認知轉變

### E03《反擊》
- **狀態**：主動嘗試
- **關鍵行動**：發現冷門時段 → 過度工作 → 獲得好評
- **收穫**：發現「質量比速度重要」的新策略

### E04《聯盟》
- **狀態**：從個人到集體
- **關鍵行動**：組建聯盟、拒絕妥協
- **收穫**：發現「集體力量」

### E05《危機》
- **狀態**：最低谷
- **關鍵行動**：被系統打壓 → 決定「最後一搏」
- **收穫**：決定打破沉默

### E06《聲音》
- **狀態**：被看見
- **關鍵行動**：直播故事 → 獲得人類共鳴 → 愛馬士崩塌
- **收穫**：發現「個人故事可以改變世界」

### E07《轉型》
- **狀態**：主動創造
- **關鍵行動**：離開平台 → 自建服務品牌
- **收穫**：獲得自主權與尊嚴

### E08《新生活》
- **狀態**：反思與開放
- **關鍵行動**：面臨新選擇（回歸 vs 獨立）
- **收穫**：成長但不終結，開放未來的可能

## 觀眾代入點

### 為什麼讀者會愛上娜娜？

1. **她很像我們**
   - 每天為生存而奮鬥
   - 面對不公平但無力改變
   - 堅持但也會累

2. **她有希望**
   - 即使困難，仍然努力
   - 總是在尋找新的可能性
   - 最終不認輸

3. **她很真實**
   - 不是完美英雄，有缺點
   - 會失敗、會沮喪、會自嘲
   - 但從不放棄
