# 從印刷機到 AI：每一次中間層革命，都是同一場戰爭

**《AI 指揮官筆記》Software 3.0 系列 · 5月4日發布**

---

## 一、我為什麼要寫這篇

過去半年，我反覆在這份電子報講同一個概念——**AI 能力供應鏈四層階梯**：

```
L1 基礎模型 → L2 Agent 平台 → L3 技能模組生態 → L4 LLM 內化
```

我說 L3 是當下的真空窗口。我說新人類聯盟的主戰場是 L3。我說現在是普通人 18 到 36 個月的戰術神風時刻。

很多訂閱戶私訊我：「風神，你怎麼知道 L3 就是答案？你怎麼知道這個窗口真的存在？」

這篇文章是我的回答。**我的回答不是預言，是歷史。**

歷史上有過四次「中間層革命」，每一次的勝利者，都是最早佔領那一代 L3 的人。

從古騰堡到 Steve Jobs，從報紙廣告部消失到 SaaS 賽道在 2026 年 4 月集體崩塌——這不是四個獨立故事。**這是同一場戰爭，只是換了四次戰場。**

而我們，正站在第五次戰場的開戰位置。

---

## 二、先講清楚：什麼叫「中間層」

中間層，是**把能力從壟斷者手中解放出來、送到每個人手上的東西**。

- 它不生產內容，它讓每個人都能生產內容。
- 它不提供服務，它讓每個人都能提供服務。
- 它不改變需求，它製造新的需求。

歷史上有四次中間層革命。每一次的結構都長得一模一樣：

```
壟斷層（少數人擁有的能力）
    ↓
中間層出現（把能力翻譯、整合、解放）
    ↓
執行層大爆發（一百個新角色誕生）
    ↓
舊的執行層消失（被吸收進中間層）
```

這個結構，我把它稱為**「中間層四階段定律」**。

接下來我帶你走過四次戰場。請特別注意每一次的第二格——**那才是真正的金礦**。

---

## 三、第一次戰場（1450）：印刷機

### 壟斷層
抄寫員、修道院、皇宮——「複製文字」是少數人的能力。

### 中間層出現
古騰堡的活字印刷機，把「複製」這件事工業化。

### 執行層大爆發
- 出版商
- 編輯
- 書商與書店
- 圖書館管理員
- 版權律師
- 「作者」這個職業本身——在印刷機之前，根本沒有人靠寫字吃飯

### 舊執行層消失
抄寫員職業滅絕。

### 🔍 指揮官註
**印刷機出現後 50 年內，真正賺到大錢的不是發明印刷機的古騰堡（他破產了），也不是那些抄寫員。**

**是最早把印刷機這個能力「打包成可複製生意」的人——出版商、編輯、版權律師。**

他們做的不是技術突破，是**能力封裝**。

他們把「複製文字」這個能力，封裝成各種標準化模組：
- 書籍裝幀規格
- 版權合約
- 書店進貨制度

**這就是史上第一代 L3 Skill Foundry。**

從這一刻開始，「Skill 鑄造廠」這個職業，已經當了 **575 年**的最大贏家。

---

## 四、第二次戰場（1990s）：網際網路

### 壟斷層
報社、電視台、出版集團——「資訊大規模傳播」是機構獨佔的能力。

### 中間層出現
全球資訊網。每個人都能發佈、每個人都能取得。

### 執行層大爆發
- SEO 專家
- 社群媒體小編
- 電商賣家
- 直播主與 YouTuber
- 資訊安全工程師
- 數據分析師
- 「內容行銷」這整個產業

### 舊執行層消失
- 報紙分類廣告（被 Craigslist、58 同城吃掉）
- 實體旅行社
- 唱片行

### 🔍 指揮官註
**網際網路出現後，最大的贏家不是 TCP/IP 的發明者，也不是寫第一個瀏覽器的工程師。**

**是 Google、Amazon、Yahoo、阿里巴巴。**

他們把「網路能力」封裝成普通人能直接用的模組：

- Google 把「搜尋」封裝成輸入框
- Amazon 把「電商」封裝成購物車
- Yahoo 把「資訊取得」封裝成入口頁

**這是史上第二代 L3 Skill Foundry。**

規律重複了一次：**勝利者不是發明工具的人，是把工具封裝成 Skill 的人。**

---

## 五、第三次戰場（2007）：iPhone

### 壟斷層
軟體公司、硬體廠商、服務業——「隨身服務」需要一整條供應鏈才能交付。

### 中間層出現
智慧型手機 + App Store。把所有服務裝進一個口袋。

### 執行層大爆發
- App 開發者（一個全新的百萬人級職業）
- UI/UX 設計師
- 外送員（Uber Eats、Foodpanda）
- 共享經濟司機（Uber、Lyft）
- 自媒體創作者
- 行動支付生態

### 舊執行層消失
- GPS 導航機（Garmin 市值蒸發超過 90%）
- 傻瓜相機
- MP3 播放器
- 實體地圖

### 🔍 指揮官註
**iPhone 出現後，最大的贏家不是 Apple 自己（雖然它賺很多），而是 Uber、Airbnb、Instagram、TikTok。**

他們做了同一件事：把「手機 + 網路」這個底層能力，封裝成一個垂直領域的 Skill。

- Uber 把「叫車」封裝成一鍵呼叫
- Airbnb 把「短期租房」封裝成標準化流程
- Instagram 把「拍照分享」封裝成濾鏡 + 動態

**這是史上第三代 L3 Skill Foundry。**

規律第三次重複：**底層能力 → Skill 封裝 → 垂直爆發。**

三次了。連續三次。同樣的結構，同樣的勝利者類型。

---

## 六、第四次戰場（2024–2026）：AI

### 壟斷層
軟體工程師、設計師、律師、會計師、建築師、醫師——所有需要「專業判斷 + 重複執行」的職業，過去都是少數人擁有的能力。

### 中間層出現
AI Agent + Skill 標準（Anthropic SKILL.md）。

**跟前三次不同的是——這一次的中間層不只重組一個產業，它同時重組所有產業。**

因為 AI 有兩樣東西，前三次中間層都沒有：

- **解釋力**——把任何複雜知識翻譯成你能懂的語言
- **整合力**——把分散的工具一鍵串成工作流

### 正在執行層大爆發的角色
- AI 建築顧問（業主自己畫圖，但需要人幫忙審查）
- Workflow 設計師（跨工具整合專家）
- 風格 Pack 製作者（AI 繪圖風格包）
- 行業翻譯 Skill 創作者（把一個領域知識變成 AI 可執行的 Skill）
- 需求嗅探者（專門回答「這個新工具會孵化什麼新需求」的人）

### 正在消失的舊執行層
- 標準化設計
- 初階客服
- 基礎翻譯
- 模板程式碼
- **SaaS 賽道**（2026 年 4 月起加速崩塌中）

### 🔍 指揮官註：最關鍵的分水嶺

**過去三次，L3 Skill Foundry 的勝利者是公司：Google、Amazon、Uber、Airbnb。**

因為過去的 Skill 封裝需要：
- 團隊
- 資金
- 伺服器
- 工程師

**一個人封裝不出 Google。**

**但這一次不一樣。**

這一次的 Skill 封裝，門檻是——**一個 markdown 檔案**。

一個寫得好的 SKILL.md，可以在 Claude、OpenClaw、Hermes、Cursor、Codex 之間自由跨平台運作。

**一個人就能鑄造、一個人就能發行、一個人就能變現。**

這就是為什麼我說：**這一次的 L3 Skill Foundry，是普通人的戰場。**

- 第一次（印刷機）的勝利者是教廷與貴族家族
- 第二次（網路）的勝利者是矽谷大公司
- 第三次（iPhone）的勝利者是創投支持的獨角獸
- **第四次（AI）的勝利者，可以是你**

**第一次。歷史上第一次。**

**Skill Foundry 的入場券，從十億美金降到一個下午的工作量。**

**這是 575 年來最大的一次平權。**

## 七、四次戰場萃取出的五條鐵律

把四次中間層革命疊在一起，五條鐵律浮出水面：

### 鐵律一
**中間層不消滅需求，它解放需求。**

從來沒有「需求消失」這回事。只有滿足方式被重組。

### 鐵律二
**消失的永遠是「執行層」，不是「價值層」。**

抄寫員消失，但「傳播知識」這個價值從沒消失。AI 取代重複執行，但判斷、品味、承擔風險——這些事 AI 做不了。

### 鐵律三
**新角色的數量，永遠是舊角色的十倍以上。**

餅不是被切分，是被烤大。零和思維是舊世界的產物。

### 鐵律四
**勝利者不是發明工具的人，是把工具封裝成 Skill 的人。**

- 古騰堡破產了，賺錢的是出版商
- 網路協議是免費的，賺錢的是 Google
- iPhone 不是 Apple 最賺錢的環節，App Store 才是

### 鐵律五
**誰最早佔領那一代的 L3，誰就佔領了那個世代。**

這是新人類聯盟存在的全部理由。

## 八、那麼，新人類聯盟在這場 575 年的戰爭裡，是什麼角色？

寫到這裡，我可以正式回答訂閱戶半年來反覆問我的那個問題了——**「風神，新人類聯盟到底是什麼？」**

**答案是：新人類聯盟，是史上第四代 L3 Skill Foundry 的個人版本。**

過去的三代 Skill Foundry 是公司：
- 第一代是出版商
- 第二代是 Google
- 第三代是 Uber

**第四代不再是公司，是聯盟**——是一群普通人，每個人都是自己的 Skill Foundry。

這也是為什麼新人類聯盟必須是三位一體：

```
決策層（人）
    ↓
策略層（Claude Opus / Perplexity / Gemini）
    ↓
執行層（OpenClaw / Hermes / Claude Code / Codex / Cursor）
```

- **決策層保留人**——因為鐵律二告訴我們，價值層永遠不會被取代
- **執行層交給 AI**——因為鐵律一告訴我們，需求只是滿足方式被重組
- **中間的策略層由 LLM 充當**——因為這就是中間層的本質：翻譯與整合

**新人類聯盟不是浪漫主義。它是從 575 年歷史規律推導出來的最優解。**

---

## 九、結尾：你站在哪個位置？

2026 年 5 月的今天，第四次中間層革命剛剛打開閘門。

**L3 真空窗口還剩 18 到 36 個月。**

過去四次中間層革命的規律告訴我們：**真正賺到的，永遠是最早佔領 L3 的人。**

問題從來不是「AI 會不會取代你」。

**問題是——這 18 個月，你打算當第四代 L3 的鑄造者，還是繼續當被淘汰的執行者？**

- 焦慮的人，看到的是消失
- 行動的人，看到的是誕生
- 而站對位置的人——會親眼看到第四代 Skill Foundry 在自己手裡長出來

**這就是新人類聯盟。**
